# Rossmann Retail Promotion Analysis

## Overview: 
This project analyses data from Rossmann stores to evaluate promotional campaign performance across store types and measuring the impact of promotions on both revenue and profit margins.

## Business Question: Are promotions actually profitable, and where are they underperforming?
- Which store types benefit most from promotions?
- Do promotions increase profit margins or only revenue?
- Which days of the week see the strongest promotional response?
- Which store types have weak promo lift and need further review? 

## Key Findings: 
- Promotions increase average sales by 39% £5,907 baseline - £8,205 promo but average margin falls from £1,772 to £1,641
- Store Type A has the strongest promotional response at 45.81% lift, Store Type B the weakest at 19.06%
- Promotions outperform baseline sales Monday–Friday; baseline holds up better at weekends
- Store Type B underperforms every other store type by a significant margin (next lowest is Store C at 35.46%)

## Business Reccomendation:
Store Type A responds strongly to promotions and should remain a priority for campaign spend. Store Type B averages only 19.06% lift, less than half of Store Type A and warrants further investigation.
